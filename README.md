# Online Quiz Management System

## Project Description

Online Quiz Management System is a React-based web application that helps students take quizzes based on class, subject, topic and difficulty level. Teachers can monitor student performance, assign homeworkk and view leaderboards.

## Components

### Signup Page

* Student registration

### Login Page

* Student login authentication

### Dashboard

* Welcome page
* Browse classes

### Subject Selection

* Select subject

### Topic Selection

* Select topic

### Difficulty Selection

* Easy, Medium, Hard

### Quiz Module

* Multiple choice questions
* Timer support
* Score calculation

### Result Module

* Display quiz score

### Teacher Dashboard

* Student scores
* Leaderboard
* Homework assignment

## Technologies Used

* React JS
* JavaScript
* CSS
* Local Storage


## Project Flow

Signup → Login → Dashboard → Subject → Topic → Difficulty → Quiz → Result → Teacher Dashboard
